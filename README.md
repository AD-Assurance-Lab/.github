# Welcome to the Automated Driving Assurance Lab Workspace 

This organization hosts the neural network verification architectures, safety assurance frameworks, and simulation-free validation tools developed at Western Michigan University.

## Repository Naming Conventions
To maintain scalability across academic tracks and future productization, all contributors must structure new repositories using our modular layout:
`[Core-Technology]-[Domain/Context]-[Variant]`

### Active Layout Examples:
* `sdp-crown-automated-driving` — Extension of the SDP-CROWN framework optimized for autonomous models.
* `tda-automated-driving` — Future tracking repository for Topological Data Analysis implementations.
